# Política de Privacidade - Amigo Secreto

**Última atualização:** 26 de fevereiro de 2024

## 1. Introdução

O aplicativo Amigo Secreto ("nós", "nosso" ou "app") respeita sua privacidade e está comprometido em proteger seus dados pessoais. Esta política de privacidade explica como coletamos, usamos e protegemos suas informações.

## 2. Informações que Coletamos

### 2.1. Informações Fornecidas Diretamente

- **Nome dos participantes**: Para organizar grupos de amigo secreto
- **Números de telefone**: Opcional, apenas se você optar por enviar SMS
- **E-mails**: Opcional, para identificação de participantes
- **Lista de desejos**: Produtos e preferências cadastrados pelos participantes

### 2.2. Informações Coletadas Automaticamente

- **Nenhuma**: O app não coleta automaticamente informações do dispositivo, localização ou dados de uso.

## 3. Como Usamos suas Informações

Usamos as informações coletadas para:

- Organizar grupos de amigo secreto
- Realizar sorteios entre participantes
- Enviar notificações por SMS (apenas se autorizado)
- Gerenciar listas de desejos dos participantes

## 4. Armazenamento de Dados

- **Todos os dados são armazenados localmente** no seu dispositivo em um banco de dados SQLite
- **Nenhum dado é enviado para servidores externos**
- Os dados permanecem apenas no seu dispositivo
- Backup automático do Android pode incluir os dados (pode ser desativado nas configurações do dispositivo)

## 5. Compartilhamento de Dados

**Nós NÃO compartilhamos seus dados com terceiros.**

O único compartilhamento de informações ocorre:
- Quando você escolhe enviar SMS manualmente aos participantes
- Quando você compartilha o resultado do sorteio através de apps de mensagens

## 6. Permissões do App

O app solicita as seguintes permissões:

### 6.1. Permissões Obrigatórias
- **VIBRATE**: Para feedback tátil em ações importantes

### 6.2. Permissões Opcionais
- **READ_CONTACTS**: Para importar contatos ao adicionar participantes (apenas se você autorizar)
- **SEND_SMS**: Para enviar automaticamente os resultados por SMS (apenas se você autorizar)
- **INTERNET**: Para possíveis atualizações futuras (atualmente não utilizado)

Todas as permissões opcionais requerem sua autorização explícita.

## 7. Segurança

Implementamos medidas de segurança para proteger seus dados:

- Comunicação apenas via HTTPS quando aplicável
- Dados armazenados localmente sem transmissão externa
- Sem coleta de dados sensíveis desnecessários
- Código ofuscado para proteção adicional

## 8. Direitos do Usuário

Você tem direito a:

- **Acessar seus dados**: Todos os dados estão visíveis no app
- **Excluir seus dados**: Use a função "Limpar" no app ou desinstale o aplicativo
- **Controlar permissões**: Ajuste nas configurações do Android em qualquer momento
- **Não fornecer dados opcionais**: Email e telefone são completamente opcionais

## 9. Dados de Menores

O app não é direcionado especificamente para menores de 13 anos, mas pode ser usado por eles com supervisão dos pais. Não coletamos intencionalmente dados de crianças.

## 10. Alterações nesta Política

Podemos atualizar esta política periodicamente. Mudanças significativas serão notificadas através de uma atualização do app. A data de "última atualização" no topo indica quando a política foi revisada.

## 11. Exclusão de Dados

Para excluir completamente seus dados:

1. Abra o app
2. Use a função "Limpar Tudo" em cada grupo
3. Ou desinstale o aplicativo

Ao desinstalar o app, todos os dados locais são automaticamente removidos do dispositivo.

## 12. Conformidade com LGPD

Este app está em conformidade com a Lei Geral de Proteção de Dados (LGPD) do Brasil:

- Coletamos apenas dados necessários para o funcionamento do app
- Não compartilhamos dados com terceiros
- Usuário tem controle total sobre seus dados
- Dados podem ser excluídos a qualquer momento
