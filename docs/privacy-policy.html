Política de Privacidade — Amigo Secreto

Última atualização: 27 de fevereiro de 2026

1. Introdução

O aplicativo Amigo Secreto ("nós", "nosso" ou "app") respeita sua privacidade e está comprometido em proteger seus dados pessoais. Esta política explica como coletamos, usamos e protegemos suas informações.

2. Informações que Coletamos

2.1. Informações fornecidas diretamente por você

  • Nome dos participantes — para organizar grupos de amigo secreto
  • Números de telefone — opcional, apenas se você optar por enviar SMS
  • E-mails — opcional, para identificação de participantes
  • Lista de desejos — produtos e preferências cadastrados pelos participantes

2.2. Informações coletadas automaticamente

Nenhuma. O app não coleta automaticamente informações do dispositivo, localização ou dados de uso.

3. Como Usamos suas Informações

  • Organizar grupos de amigo secreto
  • Realizar sorteios entre participantes
  • Abrir o aplicativo de SMS do dispositivo com a mensagem pré-preenchida (você confirma antes de enviar)
  • Gerenciar listas de desejos dos participantes

4. Armazenamento de Dados

  • Todos os dados são armazenados localmente no seu dispositivo em um banco de dados SQLite.
  • Nenhum dado é enviado para servidores externos.
  • Os dados permanecem apenas no seu dispositivo.
  • O backup automático do Android pode incluir os dados (pode ser desativado nas configurações do dispositivo).

5. Compartilhamento de Dados

Nós NÃO compartilhamos seus dados com terceiros.

O único compartilhamento de informações ocorre:

  • Quando você escolhe enviar SMS manualmente — o app abre o aplicativo de mensagens com o número e texto pré-preenchidos, mas você confirma o envio.
  • Quando você compartilha o resultado do sorteio através de apps de mensagens (WhatsApp, etc.).

6. Permissões do App

6.1. Permissões obrigatórias

  • VIBRATE — feedback tátil em ações importantes

6.2. Permissões opcionais

  • READ_CONTACTS — para importar contatos ao adicionar participantes (apenas se você autorizar)
  • INTERNET / ACCESS_NETWORK_STATE — declaradas no app, mas não utilizadas para transmissão de dados pessoais. Nenhuma informação sua é enviada a servidores externos

Todas as permissões opcionais requerem sua autorização explícita. O envio de SMS é realizado via Intent do sistema — sem necessidade de permissão SEND_SMS.

7. Segurança

  • Comunicação apenas via HTTPS quando aplicável
  • Dados armazenados localmente sem transmissão externa
  • Sem coleta de dados sensíveis desnecessários
  • Código ofuscado (R8) para proteção adicional nas versões de produção

8. Direitos do Usuário (LGPD)

Este app está em conformidade com a Lei Geral de Proteção de Dados (LGPD) do Brasil. Você tem direito a:

  • Acessar seus dados — todos os dados estão visíveis no app
  • Excluir seus dados — use a função "Limpar" no app ou desinstale o aplicativo
  • Controlar permissões — ajuste nas configurações do Android a qualquer momento
  • Não fornecer dados opcionais — email e telefone são completamente opcionais

9. Dados de Menores

O app não é direcionado especificamente para menores de 13 anos, mas pode ser usado por eles com supervisão dos pais. Não coletamos intencionalmente dados de crianças.

10. Alterações nesta Política

Podemos atualizar esta política periodicamente. Mudanças significativas serão notificadas através de uma atualização do app. A data de "última atualização" indica quando a política foi revisada.

11. Exclusão de Dados

Para excluir completamente seus dados:

  1. Abra o app
  2. Use a função "Limpar Tudo" em cada grupo
  3. Ou desinstale o aplicativo — todos os dados locais são automaticamente removidos
